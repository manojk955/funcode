Implement regex's in clojure by example

  • Regex's in clojure by example.
    • Similar to java
    • Contains 4 different constructs
      • re-matches -- matches the whole string
        user=> (re-matches #"abc" "abcdefg")
        =>nil
      • re-find -- search for string concat or regex given in input string (note it only returns first found sub string)
        user => (re-find #"s+" "fuss")
        => "ss"
      • replace -- As name suggests it replaces the sub strings with given string
        user => (clojure.string/replace "defabcdab" #"ab" "ss")
        => "defsscdss"
      • re-seq -- unlike re-find it finds all required sub strings in given input
        user =>(re-seq #"s+" "mississippi")
        ("ss" "ss")
